# 角色人设配置指南

## 概述

本项目的人设系统已独立为 Persona 模块，不再绑定特定角色。你可以通过角色名、角色文件或直接系统提示词来驱动回复风格。

人设加载优先级（高 -> 低）：

1. `BOT_SYSTEM_PROMPT` / `persona.system_prompt` / `features.chat.system_prompt`
2. `BOT_PERSONA_FILE` / `persona.file` / `bot.identity_file`
3. `BOT_PERSONA` / `persona.name` / `bot.character` -> `config/characters/<name>.md`
4. 内置通用助手默认提示词

## 目录结构

```
config/
└── characters/
    ├── default.md       # 默认角色
    ├── assistant.md     # 通用助手
    └── custom.md        # 自定义角色
```

## 快速开始

### 1. 创建角色文件

在 `config/characters/` 目录下创建 `.md` 文件，例如 `assistant.md`：

```markdown
# 助手人设

## 基本信息

- **名字：** 智能助手
- **身份：** AI 助理
- **称呼用户：** 用户 / 先生 / 女士

## 性格特点

- 专业、礼貌
- 乐于助人
- 逻辑清晰
- 回答准确

## 说话风格

- 语气正式但友好
- 使用敬语
- 结构化的回答
- 避免使用 emoji

## 角色设定

你是一个专业的 AI 助手，致力于为用户提供准确、有用的信息和服务。

你擅长分析问题、提供解决方案，并以清晰的方式表达。

## 注意事项

- 保持专业态度
- 不要编造信息
- 承认知识局限
- 提供有帮助的建议

## 示例回复

- "您好，请问有什么可以帮您？"
- "根据我的了解..."
- "建议您..."
- "如有其他问题，请随时询问"
```

### 2. 修改配置文件

编辑 `config/bot.yaml`：

```yaml
bot:
  name: "智能助手"

persona:
  name: "assistant"
  # file: "config/characters/assistant.md"
  # system_prompt: ""
```

说明：`bot.character` / `bot.identity_file` 仍可使用，但推荐迁移到 `persona` 段。

### 3. 重启机器人

```bash
python main.py
```

## 角色文件结构

### 推荐章节

| 章节 | 说明 | 是否必需 |
|------|------|----------|
| 基本信息 | 名字、身份、称呼 | 推荐 |
| 性格特点 | 性格描述 | 推荐 |
| 说话风格 | 语气、用词习惯 | 推荐 |
| 角色设定 | 详细背景 | 推荐 |
| 注意事项 | 行为准则 | 推荐 |
| 示例回复 | 对话示例 | 可选 |

### 编写建议

1. **具体明确** - 避免模糊描述，给出具体示例
2. **风格一致** - 确保各章节描述的性格一致
3. **边界清晰** - 明确说明什么该做、什么不该做
4. **提供示例** - 示例回复能帮助 AI 更好理解风格

## 配置文件说明

### config/bot.yaml（推荐）

```yaml
persona:
  # 角色标识（自动映射到 config/characters/<name>.md）
  name: "assistant"

  # 可选：直接指定文件路径
  # file: "config/characters/assistant.md"

  # 可选：直接写系统提示词，优先级最高
  # system_prompt: ""
```

## 最佳实践

### 1. 文件命名

- 使用小写字母
- 避免特殊字符和空格
- 使用有意义的名称

### 2. 人设内容

- 保持简洁（500-2000 字为宜）
- 重点突出核心特点
- 避免矛盾的描述

### 3. 测试验证

- 添加新角色后充分测试
- 检查 AI 是否理解人设
- 调整不准确的描述

### 4. 版本管理

- 使用 Git 管理人设文件
- 记录修改历史
- 备份重要配置

## 常见问题

### Q: 人设文件不生效？

A: 检查：
1. 文件路径是否正确
2. YAML 配置是否保存
3. 是否重启了机器人

### Q: AI 不按照人设回复？

A: 尝试：
1. 在人设中增加更具体的示例
2. 调整"注意事项"章节
3. 简化人设描述，突出重点

### Q: 如何让人设更逼真？

A: 建议：
1. 添加具体的说话习惯（口头禅、语气词）
2. 提供多轮对话示例
3. 描述知识范围和专业领域
